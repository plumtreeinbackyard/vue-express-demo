# server

This is a Vue/Express web application as a shopping cart demo. It uses Vuex for state management, sass for css preprocessor.

## Project setup

```
npm install
```

### development mode

```
npm run dev
```

### product mode

```
npm run start
```

## Deploy to Heroku
```
heroku login
git add .
git commit -am "make it better"
git push heroku master
```
